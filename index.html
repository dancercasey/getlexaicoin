How to Buy LEX

To purchase LEX Coin, you’ll need to set up a trustline on the Stellar network.

  1. Visit Stellarterm.
  2. Log in with your Stellar wallet.
  3. Search for the LEX token using the issuing account: GA3DQRTXVLHWVRJ3DQQ4SVWYWL7BHOXIBHVZSRAPPTTOX2LYGGU7YWOB
  4. Click “Add Trustline” and confirm.
  5. Once added, you’ll be able to trade LEX on the Stellar DEX.

Need help? Check out our GitHub guide.
